💡 Tips Importantes

🔍 Usa tap() para debugging

No modifica los datos, perfecto para console.log

⏱️ debounceTime en búsquedas

Evita peticiones innecesarias mientras el usuario escribe

🔄 switchMap vs mergeMap

switchMap cancela, mergeMap ejecuta todas en paralelo

🛡️ Siempre usa takeUntil

Previene memory leaks al destruir el componente
