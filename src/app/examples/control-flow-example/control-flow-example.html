🎯 Control Flow en Angular 17+

Ejemplos de @if, @for, @switch, @let, @empty y más

✅ 1. @if - Renderizado Condicional

Reemplaza *ngIf. Muestra u oculta contenido según una condición. Más simple y legible que la sintaxis anterior.

Ejemplo 1: @if básico

@if (isLoggedIn) {
✅ ¡Bienvenido! Has iniciado sesión correctamente.
} @if (!isLoggedIn) {
⚠️ Por favor, inicia sesión para continuar.
} 
@if (isLoggedIn) {{ '{' }}
  <div>Bienvenido!</div>
{{ '}' }}

Ejemplo 2: @if con @else

@if (isPremium) {
⭐ Usuario Premium - Acceso total
} @else {
👤 Usuario Básico - Características limitadas
} 
@if (isPremium) {{ '{' }}
  <div>Premium</div>
{{ '}' }} @else {{ '{' }}
  <div>Básico</div>
{{ '}' }}

Ejemplo 3: @if con múltiples condiciones

@if (loadingData) {
⏳ Cargando datos del servidor...
} @else if (errorMessage) {
❌ {{ errorMessage }}
} @else {
✅ Datos cargados correctamente
} 
@if (loadingData) {{ '{' }}
  <div>Cargando...</div>
{{ '}' }} @else if (errorMessage) {{ '{' }}
  <div>Error</div>
{{ '}' }} @else {{ '{' }}
  <div>Éxito</div>
{{ '}' }}

🔄 2. @for - Iteración sobre Listas

Reemplaza *ngFor. Requiere obligatoriamente track para optimizar rendimiento. Angular usa track para identificar qué elementos cambiaron.

Lista de Productos (con @for y track)

@for (product of products; track product.id) {

{{ product.name }}

{{ product.category }}

${{ product.price }}

📦 Stock: {{ product.stock }} @if (product.stock === 0) { Agotado }

} 
@for (product of products; track product.id) {{ '{' }}
  <div>{{ '{{' }} product.name {{ '}}' }}</div>
{{ '}' }}

@for con @empty (lista vacía)

@for (item of emptyList; track $index) {
{{ item }}
} @empty {
📭 No hay elementos en la lista.
Este mensaje aparece gracias a @empty
} 
@for (item of list; track $index) {{ '{' }}
  <div>{{ '{{' }} item {{ '}}' }}</div>
{{ '}' }} @empty {{ '{' }}
  <div>Lista vacía</div>
{{ '}' }}

Variables Contextuales de @for

@for (product of products; track product.id; let idx = $index; let isFirst = $first; let isLast = $last; let isEven = $even; let isOdd = $odd; let total = $count) { }
$index $first $last $even $odd $count Producto
{{ idx }} {{ isFirst ? '✅' : '❌' }} {{ isLast ? '✅' : '❌' }} {{ isEven ? '✅' : '❌' }} {{ isOdd ? '✅' : '❌' }} {{ total }} {{ product.name }} 
@for (item of list; track item.id; let i = $index) {{ '{' }}
  <div>Índice: {{ '{{' }} i {{ '}}' }}</div>
{{ '}' }}

Variables: $index, $first, $last, $even, $odd, $count

🔀 3. @switch - Selección Múltiple

Reemplaza *ngSwitch. Útil cuando tienes múltiples casos posibles. Más limpio que múltiples @if / @else if.

Estado del Pedido (Order Status)

@switch (orderStatus) { @case ('pending') {

⏳ Pedido Pendiente

Tu pedido ha sido recibido y está en espera de confirmación.

} @case ('processing') {

⚙️ Procesando Pedido

Estamos preparando tu pedido para el envío.

} @case ('shipped') {

🚚 Pedido Enviado

Tu pedido está en camino. Recibirás un código de tracking.

} @case ('delivered') {

✅ Pedido Entregado

¡Tu pedido ha sido entregado exitosamente!

} @case ('cancelled') {

❌ Pedido Cancelado

El pedido ha sido cancelado. Se procesará el reembolso.

} @default {

❓ Estado Desconocido

No se pudo determinar el estado del pedido.

} } 
@switch (orderStatus) {{ '{' }}
  @case ('pending') {{ '{' }}
    <div>Pendiente</div>
  {{ '}' }}
  @case ('processing') {{ '{' }}
    <div>Procesando</div>
  {{ '}' }}
  @default {{ '{' }}
    <div>Desconocido</div>
  {{ '}' }}
{{ '}' }}

📌 4. @let - Variables Locales en Templates

Nueva característica de Angular 18. Permite crear variables locales en el template para evitar cálculos repetidos y mejorar la legibilidad.

Cálculo de Precio Final (con @let)

@let basePrice = complexCalculation.value; @let taxAmount = basePrice * complexCalculation.tax; @let priceWithTax = basePrice + taxAmount; @let discountAmount = priceWithTax * complexCalculation.discount; @let finalPrice = priceWithTax - discountAmount;
💰 Precio Base: ${{ basePrice }}
📊 Impuesto ({{ complexCalculation.tax * 100 }}%): +${{ taxAmount.toFixed(2) }}
💵 Precio con Impuesto: ${{ priceWithTax.toFixed(2) }}
🎁 Descuento ({{ complexCalculation.discount * 100 }}%): -${{ discountAmount.toFixed(2) }}
✅ Precio Final: ${{ finalPrice.toFixed(2) }} 
@let total = price * quantity;
@let tax = total * 0.16;
@let finalPrice = total + tax;

<div>Total: {{ '{{' }} finalPrice {{ '}}' }}</div>

🎨 5. Combinación de Control Flow

Puedes combinar @if, @for, @switch dentro de un mismo componente para crear interfaces dinámicas y complejas.

Panel de Administración por Rol

{{ getRoleIcon(currentUser.role) }} {{ currentUser.name }} - Rol: {{ currentUser.role }}

@switch (currentUser.role) { @case ('admin') {

👑 Panel de Administrador

  • ✅ Acceso total al sistema
  • ✅ Gestión de usuarios
  • ✅ Configuración avanzada
  • ✅ Reportes completos
} @case ('editor') {

✏️ Panel de Editor

  • ✅ Crear y editar contenido
  • ✅ Publicar artículos
  • ❌ No puede gestionar usuarios
} @case ('viewer') {

👀 Panel de Visualizador

  • ✅ Ver contenido
  • ❌ No puede editar
  • ❌ No puede publicar
} @case ('guest') {

🚶 Panel de Invitado

  • ⚠️ Acceso limitado
  • ⚠️ Solo contenido público
} }
